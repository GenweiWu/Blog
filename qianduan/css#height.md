为什么height:100%不生效
==

## 1、html,body设置height:100%

> html
```html
<!DOCTYPE html>
<html>
<body>
   <div class="area"></div>
</body>
</html>
```
> css
```css
.area{
  height:100%;
}
```
此时高度是0
