|          | QPS              | TPS                                     |
| -------- | ---------------- | --------------------------------------- |
| 说明     | 每秒查询率       | 服务器每秒处理的事务数                  |
| 经常用于 | 评估服务器的性能 | 软件测试结果的测量单位                  |
| 区别     | 对应某个接口     | 对应某次事务                            |
|          |                  | = TPS包括一条消息入+数据处理+一条消息出 |



### 区别：

比如用户访问一次页面，有10个请求到服务器；

则TPS对应这10个请求的性能，而QPS只是某个接口的请求性能
