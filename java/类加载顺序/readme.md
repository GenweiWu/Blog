 ## 无父类时，类的加载：
   1. 加载顺序：`静态成员变量、静态代码块` >  `成员变量，代码块` > `构造函数`
   2. 成员变量和代码块的加载顺序，取决于写的顺序

[>代码](./ClassLoadTest.java)


## 有父类时，类的加载：
```
/**
 * <pre>
 * 父类的静态成员变量
 * 父类的静态代码块
 * ----------------------
 * 子类的静态成员变量
 * 子类的静态代码块
 * ----------------------
 * 父类的成员变量
 * 父类代码块
 * ----------------------
 * 父类的构造函数
 * ----------------------
 * 子类的成员变量
 * 子类代码块
 * ----------------------
 * 子类的构造函数
 * ----------------------
 * </pre>
 */
 ```
