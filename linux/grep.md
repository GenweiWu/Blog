## 用法

```
grep [选项] [--color=auto] '搜寻字符串' filename

-i 忽略大小写
-n 显示行号
-v 反向匹配，即显示不匹配的行
-r 文件夹下递归查找

^搜索  以搜索开头
搜索$ 以搜索结尾
```
