

## while

> 循环打印数字
```bash
i=0 && while(true) do echo $i;i=$(($i+1));sleep 0.01 ;done
```

```bash
#!/bin/bash
#set -x

number=0
while [ $number -lt 10 ]; do
 echo "num:$number"
 number=$(($number+1))
done
```

## for
```bash 
#!/bin/bash

for((i=0;i<10;i++)); do
   echo "->$i"
done
```

## until
```bash
#!/bin/bash

number=0
until [ $number -gt 10 ]; do
  echo "->$number"
  number=$((number+1))
done

```
