
## 1、sed特殊字符



- 替换`/`为`\`

```bash
## 用\转义，即用\/表示/，用\\表示\
# echo '111//222' | sed 's/\//\\/g'
111\\222

## 命令中的/可以用#代替
# echo '111//222' | sed 's#\/#\\#g'
111\\222
```

- 替换`'`为`A`
```bash
## 单引号无法用\转义，只需要将两侧单引号换成双引号
# cat 1.txt
111'222'333
# sed "s/'/A/g" 1.txt
111A222A333
```

- 替换`"`为`B`
```bash
## 涉及单引号的，两侧的用双引号
# cat 2.txt
333"444"555
# sed 's/"/B/g' 2.txt
333B444B555
```


- 替换`'`为`"`
- 替换`"`为`'`
```bash
# cat 3.txt
111'222"333'444"555

## 对双引号转义，外侧用双引号
# sed "s/'/\"/g" 3.txt
111"222"333"444"555
## 对双引号转义，外侧用双引号
# sed "s/\"/'/g" 3.txt
111'222'333'444'555
```

- `1aB./cD2`修改为`2.\/df`
```bash
# cat 4.txt
111aB./cD222
# sed 's#1aB./cD2#2.\/df#' 4.txt
112./df22
# sed 's#1aB./cD2#2.\\\/df#' 4.txt
112.\/df22

```
