
## 打印所有env

```groovy
pipeline {
    agent any
    stages {
        stage('debug') {
            steps {
                //打印所有env
                echo sh(returnStdout: true, script: 'env')

                //打印所有参数param
                script {
                    params.each { param ->
                        println "${param.key} -> ${param.value} "
                    }
                }
            }
        }
    }
}
```
